## 什么是 PactKit?

PactKit 是一个治理框架，在人类与 AI 智能体之间执行 **P.A.C.T.** 治理契约。它将开发工作流、角色定义和行为规则编译为可执行的**宪章**和**剧本**，适用于 [Claude Code](https://docs.anthropic.com/en/docs/claude-code) 和 [OpenCode](https://opencode.ai)。

**名称由来**: Pact（契约）+ Kit — 治理契约工具包。

核心理念: **CODE 即法则。Data 即真相。Prompt 只是指令。AI 只做创造。**

| | 原则 | 边界 |
|---|---|---|
| **P** — Prompt | 只是指令 | 告诉 AI 怎么做 — 定义流程，不定义状态 |
| **A** — AI | 只做创造 | 格式化、摘要、语言 — 不做确定性逻辑 |
| **C** — Code | 即法则 | 确定性操作的唯一执行者 — 不可绕过 |
| **T** — Truth | Data 即真相 | 所有判断的事实基础 — 不靠记忆，不靠推测 |

## 一览

| | |
|---|---|
| **9 个 Agent** | 具有受限工具的专业化角色 |
| **11 个 Command** | 以 PDCA 为核心的工作流入口 |
| **10 个 Skill** | Visualize、Board、Scaffold、Trace、Draw、Analyze 等 |
| **8 条 Rule** | 模块化宪章规则（2 条核心 + 6 条按需加载） |

### 支持的 AI 工具

| 工具 | 格式 | 命令 |
|------|--------|---------|
| **Claude Code** | Classic | `pactkit init` |
| **OpenCode** | OpenCode | `pactkit init --format opencode` |

## 快速开始

<Tabs items={['Claude Code', 'OpenCode', 'Plugin']}>
<Tab value="Claude Code">
```bash
pip install pactkit
pactkit init
```
需要 Python 3.10+。部署到 `~/.claude/`。
</Tab>
<Tab value="OpenCode">
```bash
pip install pactkit
pactkit init --format opencode
```
需要 Python 3.10+。部署到 `~/.config/opencode/`，使用 OpenCode 原生格式。
</Tab>
<Tab value="Plugin">
```bash
# 在 Claude Code 中:
/plugin marketplace add pactkit/claude-code-plugin
/plugin install pactkit
```
无需 Python。重启 Claude Code 即可使用。
</Tab>
</Tabs>

然后开始使用命令:

```bash
/project-plan "添加用户认证"
/project-act STORY-001
/project-check
/project-done
```
