## What is PactKit?

PactKit is a governance framework that enforces the **P.A.C.T.** contract between humans and AI agents. It compiles development workflows, role definitions, and behavioral rules into executable **constitutions** and **playbooks** for [Claude Code](https://docs.anthropic.com/en/docs/claude-code) and [OpenCode](https://opencode.ai).

**Name origin**: Pact (契约) + Kit — the governance contract.

The core idea: **CODE is the Law. Data is the Truth. Prompt is ONLY instruction. AI is ONLY creativity.**

| | Principle | Boundary |
|---|---|---|
| **P** — Prompt | is ONLY instruction | Tells AI how to act — defines process, never state |
| **A** — AI | is ONLY creativity | Formatting, summarization, language — never deterministic logic |
| **C** — Code | is the Law | Sole executor of deterministic operations — no bypass |
| **T** — Truth | Data is the Truth | Factual basis for all judgment — no memory, no fabrication |

## At a Glance

| | |
|---|---|
| **25 CLI Subcommands** | Deterministic operations as code, not prompts |
| **9 Agents** | Specialized roles with constrained tools |
| **11 Commands** | PDCA-centric workflow entry points |
| **10 Skills** | Visualize, Board, Scaffold, Trace, Draw, Analyze, and more |
| **8 Rules** | Modular constitution modules (2 core + 6 on-demand) |

### Supported AI Tools

| Tool | Format | Command |
|------|--------|---------|
| **Claude Code** | Classic | `pactkit init` |
| **OpenCode** | OpenCode | `pactkit init --format opencode` |

## Quick Start

<Tabs items={['Claude Code', 'OpenCode', 'Plugin']}>
<Tab value="Claude Code">
```bash
pip install pactkit
pactkit init
```
Requires Python 3.10+. Deploys to `~/.claude/`.
</Tab>
<Tab value="OpenCode">
```bash
pip install pactkit
pactkit init --format opencode
```
Requires Python 3.10+. Deploys to `~/.config/opencode/` with OpenCode-native formats.
</Tab>
<Tab value="Plugin">
```bash
# In Claude Code:
/plugin marketplace add pactkit/claude-code-plugin
/plugin install pactkit
```
No Python required. Restart Claude Code and you're ready.
</Tab>
</Tabs>

Then start using commands:

```bash
/project-plan "Add user authentication"
/project-act STORY-001
/project-check
/project-done
```
