## What is PactKit?

PactKit is an open-source, spec-driven agentic development toolkit for AI coding assistants. It compiles development workflows, role definitions, and behavioral rules into executable **constitutions** and **playbooks** for [Claude Code](https://docs.anthropic.com/en/docs/claude-code) and [OpenCode](https://opencode.ai).

**Name origin**: Pact (契约) + Kit — *Code by Contract*.

The core idea: **agree on specs before writing code, then let agents execute**.

## At a Glance

| | |
|---|---|
| **9 Agents** | Specialized roles with constrained tools |
| **11 Commands** | PDCA-centric workflow entry points |
| **10 Skills** | Visualize, Board, Scaffold, Trace, Draw, Analyze, and more |
| **8 Rules** | Modular constitution modules (2 core + 6 on-demand) |

### Supported AI Tools

| Tool | Format | Command |
|------|--------|---------|
| **Claude Code** | Classic | `pactkit init` |
| **OpenCode** | OpenCode | `pactkit init --format opencode` |

## Quick Start

<Tabs items={['Claude Code', 'OpenCode', 'Plugin']}>
<Tab value="Claude Code">
```bash
pip install pactkit
pactkit init
```
Requires Python 3.10+. Deploys to `~/.claude/`.
</Tab>
<Tab value="OpenCode">
```bash
pip install pactkit
pactkit init --format opencode
```
Requires Python 3.10+. Deploys to `~/.config/opencode/` with OpenCode-native formats.
</Tab>
<Tab value="Plugin">
```bash
# In Claude Code:
/plugin marketplace add pactkit/claude-code-plugin
/plugin install pactkit
```
No Python required. Restart Claude Code and you're ready.
</Tab>
</Tabs>

Then start using commands:

```bash
/project-plan "Add user authentication"
/project-act STORY-001
/project-check
/project-done
```
